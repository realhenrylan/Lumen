# 更新日志 (Changelog)

所有重要的版本更新都会记录在此文件中。

## [v0.6.0] - 2026-04-25

### 🎮 分辨率升级

- **2K分辨率支持**
  - 游戏分辨率从 1920×1080 升级到 2560×1440
  - 所有关卡坐标按比例缩放（×1.333）
  - 镜面尺寸相应放大，更清晰醒目
  - 目标半径增大，更容易观察
  - 障碍物尺寸同步放大

- **关卡坐标优化**
  - 入门关卡镜面：267px（之前200px）
  - 中级关卡镜面：227px（之前170px）
  - 专家关卡镜面：120px（之前90px）
  - 目标半径：37px → 16px（难度递进）

### 🎯 关卡重新设计

- **全屏元素分布**
  - 发射器不再固定在左上角
  - 发射器位置多样化：顶部、底部、左侧、右侧
  - 镜子均匀分布在画布各区域
  - 目标物体分散在整个屏幕

- **发射器位置策略**
  - 关卡1-10：左侧发射器，光线水平入射，镜子反射
  - 关卡11-20：顶部/底部发射器，光线垂直入射
  - 关卡21-50：多样化发射位置，多镜子挑战

- **光线反射路径优化**
  - 所有关卡光线路径经过物理验证
  - 确保发射器→镜子→目标的完整光路
  - 镜子位置确保光线能够击中

### 📊 技术说明

- 2K分辨率不会增加Vercel带宽消耗（Canvas渲染在客户端完成）
- 关卡加载速度不受影响（JSON大小不变）
- 桌面设备运行流畅，帧率约50-60fps
- 移动设备可能略有下降

## [v0.5.2] - 2026-04-25

### 🎨 渲染器优化

- **批量绘制优化**
  - 将同类型元素合并到单个 beginPath/stroke 调用
  - 大幅减少 Canvas 状态切换次数
  - 障碍物、镜子、目标等现在批量绘制

- **镜子端点预计算**
  - 使用 Map 缓存镜子端点（precomputeMirrorEndpoints）
  - 避免每帧重复计算 mirrorEndpoints
  - 提示模式下同时缓存当前和目标端点

- **光线追踪算法优化**
  - 使用预分配的 Point 对象（tempOrigin/tempDir等）
  - 避免频繁创建/销毁临时对象
  - 添加早退条件和高效距离计算

- **批量绘制优化**
  - 分离固定镜子、可旋转镜子、活动镜子
  - 使用统一 stroke 绘制同类元素
  - 控制点批量绘制

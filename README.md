# Lumen (V1)

一个极简的光学反射解谜原型：旋转镜面，让光束反射后命中全部目标点。

## 技术栈

- React + TypeScript + Vite
- Canvas 2D
- Zustand（游戏状态）
- Vitest（单元测试）

## 启动

```bash
npm install
npm run dev
```

## 当前实现内容（V1）

- 发射器 + 镜子 + 目标点 + 障碍物
- 鼠标/触摸旋转镜子（带操作优化）
- 实时光线追踪与反射（精确反射算法）
- 命中所有目标判定过关
- 20 关 JSON 数据驱动（全新难度梯度设计）
- 按钮：重置 / 提示 / 上一关 / 下一关
- **本地存档**（自动保存进度）
- **52个单元测试**（核心算法全覆盖）
- **移动端优化**（旋转死区、高亮反馈）
- **通关游戏冻结**（击中目标后禁止操作）
- **四周边框反射**（墙壁反弹增加策略性）
- **画布尺寸升级**（1400x800，更适合电脑操作）
- **旋转精度优化**（死区从3°降到0.5°）

## 核心模块

- `src/game/core/math.ts`：向量与反射运算
- `src/game/core/collision.ts`：射线与镜面/障碍相交
- `src/game/core/raycast.ts`：多段反射追踪与目标命中
- `src/levels/levels.json`：关卡定义
- `src/game/render/renderer.ts`：Canvas 渲染
- `src/game/input/pointer.ts`：PC/移动统一输入
- `src/game/engine/storage.ts`：本地存档系统
